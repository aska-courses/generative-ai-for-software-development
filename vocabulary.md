
# Vocabulary

  
  

| N | Word | Description |
|---|-------|-------------|
| 1 | **Generative AI** | AI that creates new content like text, images, code, or audio |
| 2 | **Prompt** | The instruction or text given to an AI model to tell it what user want |
| 3 | **Model** | The system that processes your prompt and generates an output |
| 4 | **LLM (Large Language Model)** | A type of AI trained using text data to understand and generate human-like language. |
| 5 | **API** | A method for software to request and receive data or responses from another platform. |
| 6 | **Fine-tuning** | Training an existing model further on a smaller dataset for a specific task. |
| 7 | **Inference** | The process of the AI generating an answer/output based on your prompt. |
| 8 | **Token** | The small chunks of text as an input for AI models. |
| 9 | **Context window** | The maximum size of text that the model can “remember” at one time. |
|10 | **Use-case** | A practical example of how AI is used to solve a real problem. |
|11 | **Prompt Engineering** | Writing clear prompts so the AI gives better responses. |
| 12 | **Pipeline** | Sequence of steps that AI follows to take an input (prompt/document) and produce an output (code/content/summaries).|
| 13 |  **Embedding** | A way to turn text or images into numbers so the AI can understand their meaning. |
| 14 |**Vector search** | A way to search documents by converting data into vector using embeddings, it this case it will care about text meaning rather than exact keywords. |
| 15 |**RAG** | AI searches for real information first from vector database, and then uses that information to generate an answer. |
